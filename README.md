# The Impossible Trial

A brutal boss-fighter experience for Snapchat using Snapcaht Lens Studio.

## Overview
5-8 minute combat gauntlet where players face Mellina - a three-phase boss designed to test skill and persistence.

## Project Structure
```
├── src/
│   ├── scripts/        # TypeScript/JavaScript game logic
│   ├── components/     # Reusable Lens Studio components
│   └── assets/         # Project-specific assets
├── assets/
│   ├── models/         # 3D models
│   ├── textures/       # Images, sprites
│   ├── audio/          # Sound effects, music
│   └── animations/     # Animation files
├── tests/              # Test scripts
└── docs/               # Documentation
```
